How to Run?<br>
    - Download and install MongoDB Community Server
    - Connect to Database (URI : mongodb://localhost:27017/)
    - $ git clone https://github.com/Phaswich/ClicknextXBank.git
    - $ cd ClicknextXBank
    - $ npm install
    - $ npm start
   
   Go => http://localhost:3030/
